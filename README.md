### Prerequisites
- yarn
- node
- git

### Before starting
- open covidCron.js
- change `zipcode`, `phoneNumber`, `radiusInMiles`, `filepath`, `timezone` to desired values
- time zones can be found at this [link](https://gist.github.com/diogocapela/12c6617fc87607d11fd62d2a4f42b02a)

### To start
- Run `yarn`
- to stop hit CTRL-C
